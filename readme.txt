BSPB ProgressBar allows users to set up progress bar based on shortcode in their articles.

== Description ==

BSPB ProgressBar allows users to set up progress bar based on shortcode in their articles.

= Features include: =

* Select from 2 progress bar style scripts (Horizontal, Vertical,.. more in future releases)
* Automatically extract correct shortcode from article
* Enable draw on scroll of the progress bar
* Complete settings panels admin. General, ProgressBar, Animations, Usages.

= Translations: =

* French - by [pierre gagliardi](http://pierregagliardi.com/)



== Installation ==

1. Install BSPB ProgressBar either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the BSPB Plugin settings and set your desired options.
